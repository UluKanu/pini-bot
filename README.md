# pini-bot

A simple Python script to fetch and process level data

## Features
- Fetches level data from a remote source or local file
- Processes and outputs the data in a usable format
- Easy to run with minimal dependencies

## Requirements
Install dependencies with pip:

```
pip install -r requirements.txt
```

## Usage
Run the script using:

```
python fetch_levels.py
```

## Beginner Guide: Cloning, Editing, and Pushing Changes

### Step 1: Clone the Repository
1. Open a terminal on your computer.
2. Run the following command to clone the repository:
   ```
   git clone https://github.com/itlackey/pini-bot.git
   ```
3. Navigate into the cloned directory:
   ```
   cd pini-bot
   ```

### Step 2: Open the Project in VS Code
1. Open Visual Studio Code.
2. Click on `File` > `Open Folder`.
3. Select the `pini-bot` folder you just cloned.

### Step 3: Make Changes
1. Open the file you want to edit in VS Code (e.g., `fetch_levels.py`).
2. Make your changes using the VS Code editor.

### Step 4: Commit Your Changes
1. Open the Source Control tab in VS Code (icon with three branches).
2. Stage your changes by clicking the `+` icon next to the files you modified.
3. Write a commit message in the text box (e.g., `Updated fetch_levels.py`).
4. Click the checkmark icon to commit your changes.

### Step 5: Push Your Changes to GitHub
1. Click the `...` menu in the Source Control tab.
2. Select `Push` to upload your changes to GitHub.
3. If prompted, log in to your GitHub account.

Congratulations! Your changes are now live on GitHub https://github.com/itlackey/pini-bot/
